
# CHARTVIEW

## Installation steps are -

1. git clone repo
2. npm install
3. node app.js
4. open localhost:3000


    
## Tech Stack

**Client:** Html, Css , Javascript and Bootstrap.

**Server:** Node, Express Js and Api's.
